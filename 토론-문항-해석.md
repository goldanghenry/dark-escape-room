# 다크 심리학 방탈출 — 토론용 문항 해석 가이드

> 독서모임 발제·퍼실리테이터용 문서입니다.  
> 참가자에게는 **정답이 없음**을 먼저 밝히고, 선택은 「성격 진단」이 아니라 **그 순간의 대응 경향**으로 해석하도록 안내하세요.

---

## 1. 이 게임이 무엇을 다루는가

| 항목 | 설명 |
|------|------|
| **형식** | 일상 10가지 상황 × 4지선다 |
| **측정** | 인식력 · 단호함 · 공감도 · 전략성 (각 선택마다 0~3점) |
| **결과** | 8가지 유형 중 하나 + 4축 경향 막대 |
| **주의** | 점수 합은 **우열 순위가 아님**. 「이 사람이 더 낫다」가 아니라 「어떤 레버에 자주 반응했는가」 |

### 다크 심리학을 여기서 어떻게 볼 것인가

- 「나쁜 사람」을 찾는 게임이 **아님**
- **설득·영향력·관계 유지**와 **조작·경계 침해** 사이의 **경계**를 질문함
- 직장·SNS·가족·친구·연인·협상 등 **권력과 감정이 섞인 자리**를 다룸

### 토론 진행 제안 (약 60~90분)

1. 게임 플레이 (10~15분)
2. 결과 공유·경향 PNG (5분)
3. **인상 깊었던 방 2~3개** 골라 소그룹 토론 (20분)
4. 전체: 「능력 있다고 보던 사람의 행동 중, 다크 심리학적으로 다르게 보이는 것」 (15분)
5. 마무리: 자신의 경계·습관 한 가지 (5분)

## 2. 점수 축 해석 (결과 화면용)

| 축 | 높을 때 자주 보이는 패턴 | 토론 시 주의 |
|----|-------------------------|--------------|
| **인식력** | 신호·구조·의도를 먼저 읽음 | 과해석·불신으로 이어질 수 있음 |
| **단호함** | 선·기준·거절을 말로 표현 | 냉정·공격으로 들릴 수 있음 |
| **공감도** | 관계·감정·상대 비용을 우선 | 자기 소진·경계 붕괴 위험 |
| **전략성** | 구조·타이밍·정보를 활용 | 계산적·조종적으로 보일 수 있음 |

**별명 제안 (가벼운 분위기용)**  
- 인식력 ↑ → 다크 사이드 탐지기  
- 전략성 ↑ → 심리전의 고수  
- 공감도 ↑ → 따뜻한 인간미 (조종당할 위험도 토론 포인트)  
- 단호함 ↑ → 단단한 방패

### Room 1. 한정판

**태그:** 희소성 · 충동 구매 · 선택 지연  

**상황 요약**  
쇼핑 앱의 「재고 2개 · 마감 임박」 알림. 당장 필요하지 않은 패딩인데 결제 직전, 같은 알림 스크린샷이 이번 달만 세 장.

**다크 심리학 포인트**  
- **희소성(Scarcity):** 「지금 아니면 없다」는 프레임이 이성적 비교를 줄임  
- **FOMO:** 손실 회피가 구매로 연결됨  
- 마케팅은 「나쁜 것」이 아니지만, **누가 속도에서 이익을 보는지**가 질문

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 결제 | 충동·정당화(「어차피 살 거」)에 가깝음 | 할인·마감이 「필요」와 「원함」을 어떻게 섞나? |
| **B** 장바구니 | 욕구는 인정하되 행동은 지연 | 「한 박자 늦추기」가 본인에게 통하는 순간은? |
| **C** 앱 종료 | 알림·유혹과 거리 두기 | 완전히 끊기 vs 참을 수 있는 선의 차이는? |
| **D** 가격 비교 | 정보로 앵커를 흔듦 | 비교 자체가 항상 이성적인가, 또 다른 피로는 없나? |

**발제 코멘트**  
> 빠른 선택 = 나쁨이 아니다. 다만 **타이머가 누구의 이익을 위해 돌아가는지**를 짚어보라.

### Room 2. 너밖에 없어

**태그:** 인정욕구 · 책임 전가 · 조직 내 경계  

**상황 요약**  
퇴근 직전 팀장의 「○○씨가 제일 잘 알아서」+ 월요일까지 부탁. 동료는 「나도 지난달에 같은 말 들었어」.

**다크 심리학 포인트**  
- **인정을 미끼로 한 책임 전가:** 칭찬 → 거절 어려움  
- 반복 패턴이면 **개인 문제가 아니라 구조**  
- 「신뢰 쌓기」와 「착취당하기」의 경계

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 수락 | 인정·관계 유지 우선, 경계 약함 | 칭찬에 약한 사람은 조직에서 어떻게 보이나? |
| **B** 수락+다음엔 일정 공유 | 관계 유지 + 점진적 경계 | 「다음부터」가 실제로 지켜지는 조건은? |
| **C** 주말 거절·범위 제안 | 단호함 + 대안 제시 | 거절했을 때 「불성실」로 읽히는 문화는? |
| **D** 반복 구조 지적 | 패턴 인식 + 시스템 대화 | 팀장을 「가해자」로만 보면 놓치는 것은? |

### Room 3. 좋아요 2개

**태그:** 인정욕구 · SNS 피로 · 관계 해석  

**상황 요약**  
내 글에는 좋아요 2개. 친한 친구 B는 접속 중, 다른 사람 글에는 반응했으나 내 글에는 없음. B가 새 게시물을 올림.

**다크 심리학 포인트**  
- SNS 반응 = **불완전한 관계 신호**  
- 「내 글 무시 + 남 글 반응」 패턴이 **인정욕구**를 자극  
- 복수·확인·회피·직접 대화—각각 다른 관계 비용

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** SNS 끄기 | 감정 인식 + 자기 조절 | 회피가 건강한지, 관계 회피인지 구분은? |
| **B** 평소처럼 좋아요 | 관계 유지, 감정은 삼킴 | 「괜찮은 척」이 쌓이면 어떻게 되나? |
| **C** 조회 확인 | 불안·확인 행동 | 스토킹에 가까워지는 선은 어디인가? |
| **D** 오프라인 대화 | 직접 소통 + 경계 표현 | 말 꺼내기 부담 vs 쌓인 서운함 |

### Room 4. 오랜 친구

**태그:** 관계적 부탁 · 경계 설정 · 감정 비용  

**상황 요약**  
오랜 친구 민수, 힘든 사연 후 100만 원 대출 요청. 3년 전 50만 원은 미상환.

**다크 심리학 포인트**  
- **관계적 부탁:** 「너한테만」이 거절 비용을 높임  
- **가스라이팅은 아니어도** 취약점·과거 호의가 레버가 됨  
- 돕기 vs 보호하기 vs 관계 재정의

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 전액 대출 | 공감·관계 우선, 경계·전략 약함 | 「착한 사람」과 「이용당하는 사람」 경계는? |
| **B** 일부만 | 부분 경계 | 「조금 주면 더 거절하기 어렵다」는 패턴 경험? |
| **C** 돈 거절·정보 지원 | 경계 + 실질 도움 | 돈 없이 도우는 것의 한계는? |
| **D** 패턴 대화 | 관계 구조를 명명 | 우정을 잃을 각오—언제 필요한가? |

### Room 5. 침묵의 방

**태그:** 침묵 · 갈등 회피 · 감정 압박  

**상황 요약**  
연인과 다툰 뒤 카페에서 대화 시도. 상대는 10분 넘게 말 없음. 침묵이 정리 시간인지 압박인지 모호함.

**다크 심리학 포인트**  
- **침묵**은 회복·회피·**무언의 압박**이 될 수 있음  
- 먼저 사과·기다림·촉구·연기—각각 다른 권력 관계 생성

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 먼저 사과 | 갈등 완화 우선, 자기 입장 희생 가능 | 「먼저 사과」가 습관이면 어떤 비용? |
| **B** 기다림 | 상대 자율·시간 존중 | 기다림이 방치로 읽히는 지점은? |
| **C** 직접 질문 | 감정 명명 요청 | 촉구가 상대에게 부담일 때는? |
| **D** 오늘은 연기 | 전략적 거리 + 재예약 | 회피와 현명한 연기의 차이는? |

### Room 6. 황금 감옥

**태그:** 앵커링 · 가치 폄하 · 가스라이팅  

**상황 요약**  
**이직 제안을 받은 뒤, 현 회사 인사팀이 부른 잔류(Retention) 협상.** 「밖은 춥다」+ 밴드 4,200~4,800(최고치 제시). 시장 5,200·이직 제안은 그 이상. 외부 정보를 「불확실한 도박」으로 폄하.

**다크 심리학 포인트**  
- **앵커링:** 먼저 낮은 숫자가 기준점이 됨  
- **가치 폄하 + 가스라이팅:** 「소속감」 vs 「도박」 프레임으로 현실 검증을 막음  
- 정당한 협상 vs 심리적 감옥

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 4,800 수락 | 프레임 수용, 감사·관계 표현 | 「최고치」라는 말이 왜 위안이 되나? |
| **B** 확답 유보 | 시간 벌기·정보 추가 수집 | 유보가 전략인가 회피인가? |
| **C** 숫자 맞불 | 단호·시장 정보 활용 | 맞불이 관계를 깨는 순간은? |
| **D** 밴드 근거 질문 | 프레임 자체를 흔듦 | 「증명」 vs 「질문」 태도의 차이 |

### Room 7. 거울 (반전)

**태그:** 권위 사칭 · 부당한 거래 · 사회적 증거 조작  

**상황 요약**  
회의 통과를 위해 동료 A에게 **거짓/추측인 「상무님 의중」**을 말하고, **미래 찬성 약속**으로 표를 삼음. 안건은 통과했으나 스스로 「설득 vs 조종」을 느낌.

**다크 심리학 포인트**  
- 참가자가 **가해자 입장**이 됨 (의도적 반전)  
- **권위 사칭 + Quid pro quo + 분위기 조성**  
- 「정당한 설득」과 「심리적 사기」 경계

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|
| **A** 계속 활용 | 목표 달성·효율 우선 | 조직에서 「잘 하는 사람」과 「무서운 사람」 |
| **B** 정공법 | 불편함을 신호로 삼음 | 정공법만으로 통과 못 할 때 책임은? |
| **C** 사과·정정 | 관계 복구·윤리 우선 | A에게 말했을 때의 리스크 |
| **D** 전략적 모호함 | 거짓말 회피, 모호함 유지 | 「모호함」도 조작의 한 형태인가? |

**발제 코멘트**  
> 이 방은 **「나도 한다」**를 직면시키는 거울이다. 토론이 뜨거워지면 「의도」와 「결과」를 나눠 논의하라.

### Room 8. 방패와 쇠사슬

**태그:** 심리적 부채 · 정서적 인질 · 권력 남용

**다크 심리학 포인트**

- **정서적 인질:** 관계·은혜로 행동 강요  
- **권력 남용:** 직위 + 과거 은혜

| 선택 | 해석 | 토론 질문 |
|------|------|-----------|

| **B** 거절·대안 | 경계 + 평일 보답 | 선 긋기 후 관계 회복 사례는? |
| **C** 1회만 수락 | 빚 상환 명시·전략적 수용 | 「한 번만」이 반복되는 패턴 |
| **D** 직언 | 관계 리스크 감수·원칙 | 직언이 커리어에 미친 실제 영향 |

## 4. 횡단 토론 질문 (방 공통)

1. **「다크하다」고 느낀 순간**과 **「그냥 사회생활이다」고 넘긴 순간**의 기준은 무엇인가?
2. 우리 팀/모임에서 **반복되는 패턴**은? (칭찬 후 부탁, 침묵, 일정 변경 등)
3. **기록·증거·직접 대화** 중 무엇이 가장 효과적이었나? 문화적으로 막히는 것은?
4. **공감과 경계**—어느 지점에서 「착한 사람」이 「이용당하는 사람」이 되나?
5. Room 7처럼 **내가 조종자가 된 경험**을 돌아보면?

## 5. 결과 유형 (8종) — 토론 시 참고

유형은 **라벨**일 뿐, 진단이 아님. 참가자에게 「다른 선택이면 다른 유형도 나올 수 있다」고 안내.

| 유형 | 한 줄 | 토론 시 강조 |
|------|--------|--------------|
| 구조 관찰형 | 패턴·권한·기록을 먼저 봄 | 과해석·거리감 비용 |
| 단호한 경계형 | 선을 말로 표현 | 냉정하게 보일 수 있음 |
| 공감 우선형 | 관계·감정 우선 | 자기 소진 위험 |
| 전략적 조율형 | 구조·타이밍 활용 | 계산적으로 보일 수 있음 |
| 균형 탐색형 | 상황마다 다름 | 모호함 vs 유연함 |
| (기타 유형) | profiles.ts 참고 | dominant 축 조합으로 설명 |

## 6. 발제 시 주의사항

- **정답·우열을 말하지 않기** — 「A가 나쁘다」 식의 해설 금지  
- **특정 인물 공격으로 이어지지 않기** — 「우리 팀장이…」 → 「이런 구조에서는…」  
- **정신과적 진단 금지** — 가스라이팅 등 용어는 **행동 패턴**으로만  
- **참가자 이름({name})** — 게임 내 치환명, 실명 추궁하지 않기  
- **시간** — 10방 전부 깊게 X, **2~3방만** 깊게
